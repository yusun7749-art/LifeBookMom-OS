# 생활백서맘 Factory MASTER MEMORY

## 목적
이 문서는 전체 대화를 저장하지 않는다.
프로젝트 진행 중 다음 세션에서도 이어가기 위해 필요한 맥락만 저장한다.

## 저장 대상
- 확정된 결정
- 중요한 방향 변경
- 반복하면 안 되는 실패
- 현재 진행 위치
- 다음 작업
- 장기 LOCK
- 아이디어 보관

## 저장하지 않는 것
- 대화 전문
- 생성 콘텐츠 전문
- 테스트 출력 전문
- 임시 실행 로그 전문

## RECENT CONTEXT

- 생활백서맘 Factory와 Savingio는 분리 운영한다.
- MASTER_DECISION_LOG는 결정과 검증 기록용으로 유지한다.
- 별도로 세션 기억 시스템이 필요하다.
- 목표는 작업 중 발생한 중요한 맥락을 잃지 않고 다음 세션에서 이어가는 것이다.

---

## ISSUE MEMORY

- 작업 범위가 변경될 때 기존 목표를 잊고 다른 개발 방향으로 이동하는 문제가 발생했다.
- 앞으로 작업 시작 전 MASTER MEMORY를 확인한다.
