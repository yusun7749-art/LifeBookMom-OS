# Project024.2 Korean UI

## LOCK-007 쉬운 용어 원칙
운영본부는 개발자가 아니라 운영자가 사용하는 프로그램이다.

## 적용
- Enterprise → 운영본부
- Dashboard → 운영현황
- Navigator → 오늘 할 일
- Content Studio → 글쓰기
- CMS Hub → 콘텐츠창고
- Analytics → 통계
- Revenue → 수익관리
- Settings → 환경설정
- Workflow → 작업순서
- READY → 사용 가능
- BUILDING → 개발 중
- DONE → 완료
- LOCKED → 잠금

## 기준
초등학생 학부모도 10초 안에 이해할 수 있는 화면.
