# 생활백서맘 MASTER LEARNING LOG

이 문서는 생활백서맘 자동화 프로젝트에서 발생한 **성공, 실패, 승인, 보류, 아이디어, 재발 방지 규칙**을 모두 누적 기록하는 단일 기준 문서입니다.

## 영구 운영 규칙

1. 하나의 작업 단위가 끝날 때마다 결과를 반드시 이 문서에 기록한다.
2. 성공만 기록하지 않는다. 실패, 보류, 미검증, 아이디어도 동일하게 기록한다.
3. 실행하지 않은 작업을 완료로 기록하지 않는다.
4. 테스트하지 않은 기능을 PASS로 기록하지 않는다.
5. 같은 실수를 반복하지 않도록 실패 원인과 재발 방지 규칙을 함께 남긴다.
6. 다음 작업은 항상 이 로그를 먼저 확인한 뒤 시작한다.
7. 코드 변경은 커밋 또는 PR로 저장하고, 로그에는 관련 Sprint·PR·커밋을 연결한다.
8. 이미지·글·발행 자동화는 각각 독립적으로 PASS해야 하며, 한 영역의 성공을 전체 성공으로 확대 해석하지 않는다.

## 현재 기준 상태 — 2026-07-18

### 성공

- Blogger API를 통해 실제 비공개 초안 생성 성공.
- 생성된 Blogger 초안의 post ID 확인 성공.
- Blogger 초안 검증 엔진 구현 및 병합 완료.
- 명시적 `PUBLISH` 확인값이 필요한 안전 발행 엔진 구현 및 병합 완료.
- Windows 실행 시 Python 모듈 경로 오류 수정 완료.

### 실패 및 문제

- Blogger에 올라간 글은 생활백서맘 고유 문체와 감성 DNA가 거의 반영되지 않았다.
- 글이 정보 나열형·교과서형 문체로 생성됐다.
- 실제 생활 장면, 아이의 대사, 부모의 감정, 아이의 마음, 부모의 깨달음이 부족했다.
- 썸네일이 없었다.
- 본문 연관 이미지가 없었다.
- 세로형 10컷 인포그래픽이 없었다.
- 이미지가 Blogger 본문에 자동 삽입되지 않았다.
- 이미지 ALT 텍스트와 대표 이미지 연결이 완성되지 않았다.
- 기술적으로 글이 올라간 성공을 콘텐츠 품질까지 성공한 것으로 확대 해석할 뻔했다.

### 재발 방지 규칙

- Blogger 초안 생성 성공만으로 전체 PASS 처리하지 않는다.
- 생활백서맘 Brand QA를 통과하지 못한 글은 Blogger 초안 생성 단계로 보내지 않는다.
- 필수 이미지가 모두 준비되지 않은 글은 발행 준비 상태로 전환하지 않는다.
- 글 QA와 이미지 QA를 분리 기록한다.
- 실제 Blogger 화면에서 제목, 본문, 카테고리, 이미지, 배치까지 최종 확인한다.
- 도입부는 실제 하루의 장면과 아이의 대사로 시작해야 한다.
- 본문에는 아이의 마음, 부모의 시선 변화, 현실적인 해결 과정이 포함되어야 한다.
- 2026년 한국의 실제 초등학생·맞벌이 가정 상황을 반영해야 한다.
- 블로그 글 본문에서 캐릭터 이름 `리니`를 사용하지 않고 `아이`, `우리 아이`로 표현한다.

## Sprint020 목표 — 생활백서맘 Brand & Image Rebuild

### 글 엔진

- 생활백서맘 Brand DNA 규칙을 생성 프롬프트와 QA에 강제 적용.
- 공감 장면 → 아이 대사 → 부모 감정 → 아이 마음 → 부모 깨달음 → 현실적 해결 → 여운의 구조 고정.
- 교과서형 표현과 일반 AI 문체 감점 및 차단.
- 체크리스트, FAQ, 관련 글, 추천 섹션, SEO 요소 누락 검사.

### 이미지 엔진

- 대표 썸네일 생성 계약 정비.
- 본문 연관 이미지 생성 및 삽입 위치 계약 추가.
- 세로형 10컷 인포그래픽 생성 계약 정비.
- 공식 생활백서맘 월계관·하트 워터마크를 이미지 내부 우측 하단에 고정.
- 초등학교 3학년 긴 머리 한국 여자아이 캐릭터와 밝은 크림톤 파스텔 수채화 스타일 고정.
- ALT 텍스트, 이미지 파일명, 대표 이미지, 본문 삽입 순서 자동화.

### 발행 차단 조건

다음 중 하나라도 실패하면 Blogger 초안 생성 또는 공개 발행을 차단한다.

- 생활백서맘 Brand QA 실패
- 최소 글 품질 기준 실패
- 썸네일 누락
- 본문 연관 이미지 누락
- 10컷 인포그래픽 누락
- 공식 워터마크 확인 실패
- 캐릭터 LOCK 확인 실패
- 이미지 ALT 누락
- Blogger 최종 화면 검증 실패
