# Change Approval

Status: ACTIVE

## Purpose
LOCK 자산 및 운영 규칙 변경은 반드시 승인 절차를 거친다.

| Request ID | Date | Change | Reason | Requested By | Approved By | Status |
|------------|------|--------|--------|--------------|-------------|--------|

## Approval Rules

1. LOCK 자산은 대표 승인 없이는 변경할 수 없다.
2. 승인 없는 변경은 적용하지 않는다.
3. 승인된 변경은 반드시 DecisionLog와 VersionManager를 함께 갱신한다.
4. 변경 완료 후 실제 화면을 확인한 뒤 DONE 처리한다.
