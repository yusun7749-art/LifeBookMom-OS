# Project024.1 OS Core Refactor

## 반영 항목
- Output Gate Engine
- Style Guard
- Product Intelligence V2
- Image Guard
- Brand Assets LOCK
- Output QA

## 핵심 원칙
생활백서맘 OS는 좋은 글을 만드는 프로그램이 아니라, 생활백서맘이 아닌 결과를 출력하지 못하게 하는 운영체제이다.

## Product Intelligence V2
브랜드 추천이 아니라 해결책 카테고리 추천으로 전환한다.
최근 20개 글에서 반복된 상품은 제외한다.

## Image Guard
공식 생활백서맘 월계관 워터마크만 사용한다.
새 서명, 새 로고, 위치 변경, 비율 변경을 금지한다.
