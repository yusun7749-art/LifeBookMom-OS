# LifeBookMom OS Constitution
Version: OS v2.0.0
Status: LOCKED

# 운영 목적
LifeBookMom OS의 목적은 선장님의 작업 시간을 줄이고,
브랜드 자산과 콘텐츠 생산성을 지속적으로 높이는 것이다.

# 최상위 원칙
1. GitHub는 기준 저장소이다.
2. VS Code는 작업 공간이다.
3. LOCK 자산은 대표 승인 없이는 변경할 수 없다.
4. 기존 규칙은 삭제하지 않고 ARCHIVED로 이동한다.
5. 완료는 실제 화면 확인 후 DONE 처리한다.
6. 모든 변경은 VersionManager와 DecisionLog를 함께 갱신한다.
